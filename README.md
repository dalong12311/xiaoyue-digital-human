# 小月数智人

小月数智人项目 - 一个现代化的AI交互界面

## 项目介绍

小月数智人是一个现代化的前端界面，允许用户通过浏览器与AI助手进行交互。该项目专为跨平台访问设计，支持桌面和移动设备。

## 技术栈

- **前端**: HTML, CSS, JavaScript
- **后端**: Node.js (Vercel Functions)
- **部署**: Vercel
- **版本控制**: GitHub

## 功能特性

- 现代化UI设计
- 实时聊天功能
- 响应式布局
- 跨平台兼容

## 部署

本项目可直接部署到Vercel平台。

## 项目结构

- `index.html` - 主界面
- `api/chat.js` - 聊天API接口
- `api/backend_connect.js` - AI核心连接
- `package.json` - 项目配置
- `vercel.json` - Vercel配置

## 安全说明

本项目使用密码保护机制，确保数据安全。
